# SportZone Virtual Card Gateway

Plugin WooCommerce tao cong thanh toan the ao cho demo/do an.

## Cach cai

1. Copy thu muc `sportzone-virtual-card-gateway` vao `wp-content/plugins/`.
2. Vao WordPress Admin > Plugins va kich hoat `SportZone Virtual Card Gateway`.
3. Vao WooCommerce > Settings > Payments.
4. Bat phuong thuc `Thanh toan bang the ao`.

## The demo

- So the: `4242 4242 4242 4242`
- CVV: `123`
- Ngay het han: bat ky ngay tuong lai, vi du `12/30`

Plugin khong luu so the va khong xu ly giao dich ngan hang that.
