# SportZone WordPress

Website ban do the thao xay dung bang WordPress, PHP va WooCommerce.

## Thanh phan

- `sportzone-theme`: theme giao dien cua website.
- `sportzone-demo-content`: plugin tao trang, menu, danh muc va san pham mau.
- `sportzone-virtual-card-gateway`: plugin thanh toan bang the ao cho demo.

## Yeu cau

- XAMPP hoac moi truong PHP/MySQL tuong duong.
- WordPress.
- WooCommerce.

## Cai dat nhanh

1. Copy `sportzone-theme` vao `wp-content/themes/`.
2. Copy `sportzone-demo-content` va `sportzone-virtual-card-gateway` vao `wp-content/plugins/`.
3. Kich hoat theme `SportZone Shop`.
4. Kich hoat WooCommerce.
5. Kich hoat `SportZone Demo Content`.
6. Kich hoat `SportZone Virtual Card Gateway`.

Chi tiet xem file `GITHUB_DEPLOY.md`.
